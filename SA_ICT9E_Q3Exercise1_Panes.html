HTML

HyperText Markup Language

With HTML, you can create your very own website. It is a the standard markup language for webpages. Moreover, it is like the "hidden code" that helps us communicate.

CSS

Cascading Style Sheets

CSS are the HTML elements that you see on your screen. It can make your webpages more interesting to look at. Plus, it allows you to be creative with how you design your website.

Bootstrap

Bootstrap Version 4

Bootstrap allows you to easily create responsive designs. Responsive web designs automatiaclly adjusts its content based on the screen you're viewing it in. These designs make your website look more put together.
